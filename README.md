# Virtual CV

A concise portfolio site highlighting my skills, experience, and contact details.

---

## Quick Start

1. **Clone**  
   `git clone https://github.com/Bernard1906/virtual-cv.git`

2. **Add Your Photo**  
   Put `profile.jpg` into the `assets/` folder.

3. **Edit**  
   - Update text in `index.html`  
   - Tweak styles in `styles.css`  
   - Adjust animations in `script.js`

4. **Deploy**  
git add .
git commit -m "Update site"
git push origin main

Live at:  
https://Bernard1906.github.io/virtual-cv/
